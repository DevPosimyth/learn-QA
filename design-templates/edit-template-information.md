# How to Edit Information of a Template or Website Kit

## Overview

WDesignKit enables straightforward editing of template and website kit information from both the app and plugin.

**Requirement:** WDesignKit must be installed and activated.

---

## How to Edit

### Via WDesignKit App

1. Navigate to **Templates > Uploaded**
2. Locate your target template and click the **three-dot menu**
3. Select **Edit**
4. Make your changes and click **Save**

### Via WDesignKit Plugin

1. Go to the **Templates** section
2. Hover over your desired template
3. Click the **three-dot menu**, then **Edit**
4. You'll be redirected to the app for editing

---

## Editable Fields

| Field | Required |
|---|---|
| **Title** | ✅ Required |
| Description | Optional |
| Short Introduction | Optional |
| Features | Optional |
| Live Demo URL | Optional |
| Elementor Version URL | Optional |
| Gutenberg Version URL | Optional |
| Figma Version URL | Optional |

---

## Additional Settings

- **Visibility** — Private (only you) or Public (all WDesignKit users)
- **Publish date** — Defaults to current date if not specified
- **Required plugins** — Configure via the Manage tab
- **Template type** — Page Template or Section
- **Categories and tags** — For searchability
- **Featured image** — Upload a representative image
- **Sections and pages** — Add/remove (pagekit only)

Click **Save** to confirm all changes.
