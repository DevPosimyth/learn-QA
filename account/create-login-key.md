# How to Create Login Key to Login to WDesignKit Account?

## Overview

WDesignKit allows users to generate a Login Key for secure account access. This feature enables authorized individuals to log into your account without sharing username and password credentials, enhancing both security and convenience.

## Creating a Login Key

Follow these steps to generate a Login Key:

1. Log into your WDesignKit account within the WDesignKit app
2. Navigate to **Licence** > **Login Key** from the Dashboard
3. Enter a name for the key in the "Create a new Login Key" field
4. Click the **Generate Key** button to create your Login Key

## Using Your Login Key

Once generated, you can use the Login Key to access your account:

1. Copy the Login Key using the copy button
2. Visit the WDesignKit plugin login page
3. Click the **Connect via Login Key** button
4. Paste your Login Key in the designated field
5. Click **Log In** to access your account

**Important Note:** The Login Key functions exclusively through the WDesignKit plugin page, not other login methods.

## Deleting a Login Key

To revoke someone's access:

1. Click the delete icon next to the specific Login Key
2. Confirm deletion by clicking **Yes** in the popup dialog

This feature proves valuable when granting temporary account access for support purposes without compromising login credentials.
