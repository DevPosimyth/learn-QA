# How to Import and Export Elementor Widget or Gutenberg Block?

## Overview

WDesignKit enables seamless importing and exporting of widgets and blocks locally, without requiring server uploads. This is particularly useful for teams managing multiple websites who want to share design components across projects.

**Requirement:** WDesignKit must be installed and activated on your WordPress site.

---

## Export Widgets or Blocks

To download a widget or block locally:

1. Navigate to the WDesignKit plugin dashboard
2. Go to **Widgets > My Widgets**
3. Locate the widget or block you wish to export
4. Click the **three-dot menu icon** next to the item
5. Select **Download ZIP**
6. The widget or block downloads as a compressed file to your local system

---

## Import Widgets or Blocks

To add previously exported widgets or blocks to your site:

1. Access the WDesignKit plugin dashboard
2. Go to **Widgets > My Widgets**
3. Click the **Import Widget** button at the top of the page
4. In the popup dialog, select your downloaded widget or block ZIP file
5. Click the **Import** button
6. Your widget or block appears on the **My Widgets** page

---

## Key Benefit

This local import/export system allows designers and developers to maintain control over custom design assets while building a reusable library across multiple WordPress installations.
