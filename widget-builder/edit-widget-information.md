# How to Edit Information of an Elementor Widget or Gutenberg Block?

## Overview

WDesignKit offers a streamlined process for modifying widget and block information, allowing users to customize titles, descriptions, and other details.

**Requirement:** WDesignKit must be installed and activated. Your widget must be uploaded to the WDesignKit server before editing.

---

## Step-by-Step Instructions

### 1. Access the Widget

Navigate to **Widgets > Uploaded** in the WDesignKit app after logging in.

### 2. Open Edit Mode

Locate your desired widget or block, click the **three-dot menu**, and select **Edit**.

### 3. Edit Basic Information

| Field | Description |
|---|---|
| **Title** | Modify the widget or block name |
| **Description** | Optional detailed information |
| **Short Introduction** | Optional brief description |
| **Demo URL** | Optional preview link |
| **Tutorial URL** | Optional instructional resource link |
| **Features** | Optional section to highlight capabilities |

### 4. Add External Files

- **JS Files** — Click **+Add more JS** to include JavaScript file names and sizes (in KB)
- **CSS Files** — Click **+Add more CSS** for stylesheet details
- **jQuery Files** — Click **+Add more JQuery** for jQuery library information

### 5. Configure Visibility

Use the **Status** dropdown:

- **Private** — Only you can access; shareable within your workspace *(default)*
- **Public** — All WDesignKit users can access

### 6. Manage Additional Details (Manage Tab)

- **Page Builder** — Specify which page builder hosts your widget
- **Category** — Assign or change the widget category
- **Tags** — Add or modify associated tags
- **Featured Image** — Upload a representative image
- **Date** — Adjust the publish date

### 7. Save Changes

Click **Save** to update all modifications.
