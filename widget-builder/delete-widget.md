# How to Delete Elementor Widget or Gutenberg Block?

## Overview

Removing unnecessary widgets or blocks from your project helps manage your custom widget library efficiently. Free WDesignKit accounts have a limit of 15 custom widgets or blocks, and downloaded widgets count toward this limit.

**Requirement:** The WDesignKit plugin must be installed and activated.

---

## Delete From Your Site

1. Navigate to the WDesignKit plugin page
2. Go to **Widgets > My Widgets**
3. Locate the widget or block you want to remove
4. Click the **three dots menu icon**
5. Select **Delete**

### If Widget Is NOT Uploaded to Server

A popup appears with one option:
- Click **Local Delete**
- Confirm by clicking **Yes, Delete It!**

### If Widget IS Uploaded to Server

Two options appear (requires being logged into your WDesignKit account):

- **Permanently Delete** — Removes from both server and site
- **Local Delete** — Removes from your site only; remains on server

Select the appropriate option and confirm with **Yes, Delete It!**

---

## Delete From Server Only

To remove a widget exclusively from the server:

1. Visit the WDesignKit website and log into your account
2. Navigate to **Widgets > Uploaded**
3. Find the desired widget or block
4. Click the **three dots menu**
5. Select **Delete**
6. Confirm deletion with **Yes**

The widget is deleted from the server but remains on your site.
