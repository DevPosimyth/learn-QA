# How to Manage Versions of a Widget?

## Overview

In WDesignKit, managing widget versions allows developers to stay organized and track important updates. This feature is particularly useful for public widgets, enabling other users to view changes and updates.

**Requirement:** The WDesignKit plugin must be installed and activated.

---

## Step-by-Step Instructions

### 1. Access Your Widgets

1. Log into your WDesignKit account from the plugin page
2. Navigate to **Widgets > My Widget**

### 2. Initiate Version Push

Select the widget you want to update, click the **three dots menu**, then choose **Push Widget**.

**Alternative method:** Click **Edit in New Tab**, then select the **Push** button in the widget editor.

### 3. Configure Version Details

The **Add Sync Details** popup will appear, displaying:
- Current version number
- A field to enter the latest version number

> **Important:** Make sure to set the latest version number higher than the current one. When syncing for the first time, the latest version field remains disabled.

### 4. Add Changelogs

Click the **+Add more** button to document multiple changelog items describing your widget's changes.

### 5. Upload and Publish

Click the **Update** button to upload your widget to the server with version information.

---

## Visibility

Version information and changelogs appear on your widget's dedicated page on the WDesignKit website, allowing other users to review what changed with each release.

This feature is most valuable when making widgets publicly available.
