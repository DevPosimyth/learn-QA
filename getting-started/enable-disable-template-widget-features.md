# How to Enable or Disable Template and Widget Features in WDesignKit

## Overview

WDesignKit allows users to selectively activate or deactivate three core asset types: Design Templates, Widget Builder, and Code Snippets. Disabling features you don't use keeps your workspace focused and reduces server load.

**Requirement:** WDesignKit must be installed and activated.

---

## Steps to Manage Features

1. Navigate to the WDesignKit plugin page
2. Go to **Settings**
3. Click **Customize** in the **Features Manager** section
4. In the management interface, independently toggle:
   - **Design Template** — Enable/disable the template library
   - **Widget Builder** — Enable/disable widget creation tools
   - **Code Snippets** — Enable/disable the snippet library
5. Save your settings

---

## What Happens When You Disable a Feature

When a feature is disabled, two things occur:

1. **Interface removed** — Related options are removed from the WDesignKit plugin page
2. **Files stop loading** — Associated server files no longer load

For example, disabling Widget Builder removes all widget-related options from the plugin and stops corresponding files from loading.

---

## When to Use This

- You only use WDesignKit for templates — disable Widget Builder and Snippets
- You only use Snippets — disable Design Templates and Widget Builder
- You want a cleaner, focused dashboard for your team
