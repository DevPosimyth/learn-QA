# How to Import Templates From WDesignKit Public Library to Your Elementor or Gutenberg Site

## Overview

WDesignKit enables rapid website building through its Public Library of professionally designed templates. Users can import complete pages, sections, or website kits and customize them for their needs across both Elementor and Gutenberg platforms.

**Requirement:** The WDesignKit plugin must be installed and activated.

---

## Accessing the Template Library

Navigate to **WDesignKit → Templates → Browse Templates** from your WordPress dashboard.

### Available Filters

- Search Templates
- AI Compatible (Templates)
- Page Builder (Elementor or Gutenberg)
- Free or Pro templates
- Template Type (Page Kits, Full Pages, Sections)
- Plugins (Elementor, Elementor Pro, The Plus Addons for Elementor)

---

## Importing Page or Section Templates

### Step 1: Select the Page Builder

Choose between Elementor templates or Gutenberg templates.

### Step 2: Choose Template Type

Select either **Full Pages** (complete page layouts) or **Sections** (specific page sections).

### Step 3: Import the Template

Hover over your desired template and click the **Download icon** to begin the automatic setup process.

### Step 4: Automatic Setup Process

WDesignKit automatically handles:
- Installing and activating required plugins and themes
- Configuring necessary plugin settings
- Applying required theme settings
- Importing the selected template

> **Note:** Inactive plugins are automatically activated. Premium plugins won't install automatically — missing premium plugins may cause elements to display incorrectly.

### Step 5: View the Imported Template

Upon successful completion, click **View Template** to open and customize your imported page.

---

## Importing Website Kits

Website Kits contain complete template sets for building entire websites quickly.

### Step 1: Select Page Builder & Template Type

Choose Elementor or Gutenberg, then select **Page Kits** to display available website kits.

### Step 2: Select Kit Category

Choose from: Agency, WooCommerce, Creative Kit, Portfolio, Entertainment, Blog.

### Step 3: Start Website Kit Import

Hover over your chosen kit and click the **Download icon** to open the Website Kit Setup Wizard.

### Step 4: Enter Website Details

Provide basic information:
- Business Name
- Business Tagline
- Logo
- Optional: Address, Email, Phone Number, Social Media Links

Click **Next** to proceed.

### Step 5: Select Fonts and Color Palette

- Choose a **Font Pair**
- Select a **Color Palette**
- Add custom colors if needed

Click **Next** to continue.

### Step 6: Choose Kit Features

Select required features:
- eCommerce (for WooCommerce)
- Dynamic Content
- Performance Optimization
- Security Tools
- Extras (Forms, Popups, etc.)
- Nexter Theme (recommended)

### Step 7: Content & Media Setup

**Import Dummy Content** — Imports ready-made sample pages, text, and media matching the demo preview.

**Import as Wireframe (Layout Only)** — Imports only website structure/layout with placeholder images.

**Generate AI Content (Beta)** — Automatically creates website content using AI based on your business details.

### Step 8: Image Selection

Choose from:
- **Stock Images** — Browse built-in library
- **Lummi Images** — Explore AI-curated images
- **Upload Your Images** — Add personal files (PNG, JPEG, WebP, AVIF up to 20 MB)

Select at least one image, then click **Import**.

### Step 9: Import Process

The system automatically performs:
- Installing Plugins & Theme
- Importing Site Content
- Setting Up Pages & Layout
- Finalizing Settings

Upon completion: **Success! Your Website is Ready.**

---

## Importing From Elementor Editor

1. Open any page and click **Edit with Elementor**
2. Click the **WDesignKit icon** inside the editor
3. Navigate to the **Templates** section
4. Hover over your selection and click the **Download icon**

---

## Importing From WordPress (Gutenberg) Editor

1. Open any page using the WordPress block editor
2. Click the **Import Templates – WDesignKit** button at the top
3. Hover over your desired template
4. Click the **Download icon** to start import
