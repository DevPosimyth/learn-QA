# How to Import AI Kits in WDesignKit

## Overview

AI Kits are complete, AI-powered website kits that can be imported and customized in a 7-step process using WDesignKit.

---

## Step 1: Browse and Select a Kit

Navigate to **Templates**, filter by page builder (Elementor/Gutenberg) and free/pro status, then select your desired kit.

## Step 2: Enter Business Details

Provide your:
- Business name
- Tagline
- Logo
- Optional: address, email, phone, social media links

This data personalizes your website automatically through AI integration.

## Step 3: Design Customization

Choose:
- **Font pair** for headings and body text
- **Color palette** for your site's visual identity
- Optional custom color additions

## Step 4: Select Features

Enable the features you need:
- Dynamic Content
- Performance Optimization
- Security Tools
- eCommerce (WooCommerce)
- Extras (Forms, Popups, etc.)

## Step 5: Content Import Options

Select how content loads:

| Option | Description |
|---|---|
| **Dummy Content** | Sample pages, text, and media matching the demo |
| **AI-Generated Copy** | Content written automatically based on your business details |
| **Wireframe Only** | Layout/structure only, no dummy content |
| **AI Blog Posts** | Automatically generated blog content |

Image selection also occurs at this step (unless wireframe mode is enabled).

## Step 6: Automatic Installation

WDesignKit automatically:
- Installs required plugins
- Applies global settings and theme configurations
- Sets up pages and layouts

## Step 7: Launch

Preview your completed website and begin customizing using your preferred page builder.
