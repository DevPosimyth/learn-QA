# How to Modify Widgets Downloaded From WDesignKit's Public Library

## Overview

WDesignKit enables users to customize widgets downloaded from its public library, adjusting design elements, content, and functionality to match specific requirements.

**Requirement:** WDesignKit must be installed and activated.

---

## Steps to Modify a Downloaded Widget

1. Navigate to **Widgets > My Widget** in your dashboard
2. Locate the downloaded widget you want to edit
3. Click the **three-dot menu** next to the widget
4. Select **Edit in New Tab**
5. The widget editor opens — you can:
   - Modify the design
   - Add or remove controls
   - Edit HTML, CSS, and JavaScript code
6. Click **Save** when done

---

## Important Note

> **Keep in mind:** When you update the downloaded widget with a new version from the library, it will override your changes.

If you've made significant customizations, document them separately before updating to a new version.

---

## Also Works For

This same process works for editing your own custom-created widgets — not just downloaded ones.
