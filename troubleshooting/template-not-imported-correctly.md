# Template Not Imported Correctly or Some Elements Are Missing (Fixed)

## Overview

Users experiencing template import issues or missing elements from WDesignKit have several troubleshooting options available. The solutions differ slightly between free and pro templates, though many steps overlap.

---

## Free Templates Solutions

When importing free templates, take these steps:

- **Log into your WDesignKit account** from the WDesignKit plugin within your WordPress site
- **Install and activate all required plugins** used by the specific template you're importing
- **Enable all blocks/widgets** in the relevant plugin. For example, if using The Plus Addons for Elementor, navigate to **WordPress Dashboard > ThePlus Settings > Plus Widgets** and enable all widgets

---

## Pro Templates Solutions

For pro template imports, follow these additional steps beyond the free template checklist:

- Verify you're logged into your WDesignKit account via the WordPress plugin
- Install and activate all plugins required by the template
- **Ensure all pro plugin licenses are activated** on your WordPress site
- Enable all blocks/widgets in the relevant plugins

---

## Activate Elementor Flexbox Container

All WDesignKit templates use Elementor Flexbox containers. To activate this feature:

1. Go to **Elementor > Settings > Features**
2. Select **Active** from the **Flexbox Container** dropdown
3. Save your changes
4. Retry the template import

---

## Contact Support

If issues persist after completing all steps above, raise a support ticket at the helpdesk. When submitting, include:

- The WDesignKit template URL
- Temporary WordPress login access
- WDesignKit account login credentials or login key
