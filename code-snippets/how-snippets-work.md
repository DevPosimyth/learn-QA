# How Do Snippets Work in WDesignKit?

## Overview

The Code Snippet feature in WDesignKit allows users to create, manage, import, export, organize, and reuse custom code for WordPress websites. Rather than manually editing theme files or relying on multiple plugins, you can store and manage all code in one centralized platform.

Snippets are small, ready-to-use pieces of code or settings that add functionality or customization without requiring manual coding. They handle tasks like disabling comments, hiding WordPress version numbers, improving security, or optimizing performance.

## Key Benefits

- **No coding required** — Add advanced features without editing files
- **Safe and easy** — Snippets are pre-tested and secure to use
- **Time-saving** — Quickly apply changes to your site with one click
- **Flexible** — Use single snippets or bundles depending on your needs
- **Organized** — Manage all snippets from one dashboard

---

## How Snippets Work

The Snippets feature provides an easy way to browse, activate, and manage code snippets directly from your WordPress dashboard using WDesignKit. It includes two main options under the Snippets menu.

## Access Locations

### A) From the WDesignKit App (Online Account)

1. Log in to your WDesignKit account
2. From the top navigation menu, click **Snippets**
3. You will see two main options:
   - **Browse Snippets** — Explore ready-made snippets & bundles
   - **My Snippet** — Access your personal saved, imported, created, or downloaded snippets

### B) From the WordPress Plugin Dashboard

1. Go to WordPress Admin Dashboard
2. Navigate to **Code Snippet → Uploaded / Favourite / Downloads / Shared with Me / Create New Kit**

Inside the plugin panel, you can view all snippets, edit snippets, save to cloud, import snippets, and create new kits.

---

## Browse Snippets

When you click **Browse Snippets**, you'll see a library of all available snippets.

### Available Filters

- **Free/Pro** — Show only Free, Pro, or All snippets
- **Snippet Type** — Filter by code type
- **Category** — View snippets by purpose or feature
- **Plugins** — Filter by related plugins
- **Related Tags** — Search using keyword tags

### Snippet Types

#### 1. Individual Snippet

A single piece of code that performs a specific function.

**To download:**
1. Click on the snippet you want
2. Click the **Download** button (top right)

#### 2. Snippet Bundle

A collection of multiple snippets grouped together for a specific purpose.

**Example — WP Security Hardening bundle includes:**
- Completely disabling comments
- Disabling XML-RPC
- Removing the WordPress version number
- Disabling the plugin/theme editor

**To download a bundle:**
1. Click on the snippet bundle
2. Select the snippets you want
3. Click the **Download** button (top right)

> **Note:** To use any Snippet feature, you must have the **Nexter Extension** activated. If it's not already activated, it will be automatically activated on your site when you download a snippet.

## My Snippet

Navigate to **WDesignKit → Snippets → My Snippet**

This section allows you to:
- Add your own custom snippets
- Edit existing snippets
- Track snippets you've downloaded or activated
- Organize snippets for reuse across multiple projects
