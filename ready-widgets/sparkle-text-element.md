# Sparkle Text Element

## Overview

The Sparkle Text Element adds a soft, moving sparkle effect to any text on your website, helping important words stand out in a clean and elegant way. Fully customize sparkle size, speed, color, and density. Optimized for performance and cross-device compatibility.

## Requirements

- Bricks Builder (installed and activated)
- WDesignKit plugin (installed and activated)
- Active WDesignKit account
- Downloaded Sparkle Text element

## Activation Steps

1. Navigate to **WDesignKit → Widgets → Browse Widgets**
2. Search for "Sparkle Text" and select **Bricks** as the page builder
3. Click the **Download icon**

## Configuration Options

### Layout Tab

- **Style** — Select from pre-defined style options
- Different configuration options appear based on selected style

### Content Tab

- **Text Field** — Add the desired text content

### Icons Tab *(Available when Style 2 is selected)*

- **Select Icon** — Choose an icon or SVG element
- **Icon Color** — Manage normal icon color
- **SVG Color** — Manage SVG icon color
- **Add More** — Click **+ ADD ITEM** to add additional icons

### Extra Options Tab

- **Effect on Hover** — Enable sparkle animation to display only on hover
- **Sparkle Duration (ms)** — Control animation speed (smaller = faster; larger = slower)
- **Alignment** — Configure responsive device alignment

### Style Tab

Further customize the visual appearance of the sparkle effect.
